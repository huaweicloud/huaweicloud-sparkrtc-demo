# huaweicloud-video-Broadcasting

展示这个开源示例项目的目的是演示如何快速集成SparkRTC SDK，实现多人实时音视频。
多人实时视频直播的示例包含有Android、iOS、Windows和Web平台。
多人实时视频直播示例代码展示的功能为：
- 加入会议和离开会议；
- Join和Play之前切换角色；
- 开关本地摄像头；
- 开关本地麦克风；
- 切换本地音频路由；
- 会议Join角色成员展示；
- 切换摄像头；
- 修改远端选看分辨率。
- 大小班课；
